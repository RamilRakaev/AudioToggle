# AudioToggle
A console application that switches the computer to the next output device when launched.
When entered into Powershell/CMD, "AudioToggle.exe --list" displays the order in which it switches.

Консольное приложение, которое при запуске переключает компьютер на следующее устройство вывода.
При вводе в Powershell/CMD "AudioToggle.exe --list" выводит  порядок, в котором оно переключает.
